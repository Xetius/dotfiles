# Neovim Cheatsheet

## Install
To install the latest release of Neovim:
 ```
brew install neovim
```
To install the latest nightly of Neovim:
```
brew install --HEAD neovim
```
## Basic
h,j,k,l | Left, Down, Up, Right cursor movement.
I, i, a, A | Insert Mode Beginning of line, before cursor, after cursor, end of line
